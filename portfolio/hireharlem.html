Hire Harlem

A website that helped you find minority-owned businesses. It started in Harlem, NY and expanded to Oakland, CA. The website was created to help support the local community and to help people find businesses that they can support. Some of the features on the Hire Harlem website that I've contributed to have been

  1. A complete redesign of the website to make it more user-friendly. The new technology stack used was React.js, Redux, and Express.js.
  2. Reconstructed the back-end of the website. Moved it to MongoDB with an Express.js backend.
  3. Worked in tandem with designers, project managers, and other developers to create a more cohesive and user-friendly experience.
