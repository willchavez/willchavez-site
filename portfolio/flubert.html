Flubert

A design assistant that helps you create beautiful and consistent designs that adhere to the Fluent Design System. Flubert is a figma plugin that provides you with a set of tools to help you design faster and more consistently. Some of the features on Flubert that I've contributed to have been

  1. Incorporating iconography and emoji suggestions based on the Fluent Design System.
  2. Creating search indexes based on scraped metadata for icons and emoji.
