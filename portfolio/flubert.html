Flubert

@ Fluent Design Studio, Microsoft
about

A design assistant that helps you create beautiful and consistent designs that adhere to the Fluent Design System. Flubert is a Figma plugin that analyzes your UI against Fluent guidelines and surfaces recommendations — helping designers move faster without sacrificing consistency.

contributions
  1. built iconography and emoji suggestion features grounded in a vector database of Fluent assets, using scraped icon and emoji metadata as the retrieval corpus
  2. created search indexes from that metadata to power fast, semantically relevant suggestions within the plugin
  3. integrated LLM-based analysis to evaluate UI components against Fluent design guidelines and surface actionable feedback
