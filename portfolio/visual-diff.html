Visual Diff Tool

This tool came out of a user need from designers at Microsoft that wanted a visual representation of the differences in updated icons in the microsoft/fluentui-system-icons repo.

  1. It generates an overlayed image of the old and new icon, with coloring and transparency see to indicate the updated pixel areas.
  2. It uses imagemagick to numerically represent the visual differences based on pixel differences.
  3. The tool started off as it's own separate website, where we passed in the new icons to make the comparisons. It has since been updated to a python script that runs on our pipeline so that we're not maintaining a separate website.
