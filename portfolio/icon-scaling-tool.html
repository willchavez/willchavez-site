Icon Scaling Tool

A figma plugin used internally at Microsoft and available world-wide. Scales icons on Figma, and also exports them in PDF & SVG formats. Some of the features on the Icon Scaling tool that I've contributed to have been

  1. Making the plugin completely headless for exporting and scaling icons.
  2. Combining vector paths to reduce the number of nodes in the SVG. (Squashing)
  3. Exporting metadata such as metaphors, descriptions, sizes, and themes for each icon component.
